# My Blog — ApexPlanet Internship Project

A fully functional blog web application built with PHP & MySQL.

## Features
- CRUD (Create, Read, Update, Delete) for blog posts
- Search with keyword filter
- Pagination (3 posts per page)
- User Registration & Login
- Password hashing with bcrypt
- Role-Based Access Control (admin, editor, user)
- Admin panel to manage users
- CSRF Protection on all forms
- SQL Injection prevention with PDO
- Form validation (server-side + client-side)
- Dark mode / Light mode toggle
- Responsive design with Bootstrap 5
